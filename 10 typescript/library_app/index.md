## 도서관 관리 프로그램

- TypeScript에서 배웠던 인터페이스, 타입 별칭, 그리고 타입 가드를 활용하여 온라인 도서관 관리 프로그램

### 기능

- 온라인 도서관 관리 프로그램은 다음과 같은 기능을 제공합니다:
- **도서 등록 기능** - 관리자 (Admin)
- **도서 삭제 기능** - 관리자 (Admin)
- **도서 대출 기능** - 사용자 (User)
- **도서 반납 기능** - 사용자 (User)
- **도서 검색/조회 기능**
- 관리자, 사용자 (제목, 저자, ISBN 등으로 검색)
- 회원 등록/조회 기능 - 관리자
